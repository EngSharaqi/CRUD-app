# V.Sharaqi admin CRUD dashboard
#### WELOCME

#### Run this application Locally:
1. ```git clone https://github.com/EngSharaqi/CRUD-app.git```
2. ```yarn install```
3. ```npm install```
4. Go to browser the visit this link [http://localhost:3000/](http://localhost:3000/)
